Deteccion de OSes con TTL y Python

Python, TTL, Hacking

En mi primer blog estare hablando sobre el TTL y como reconocer sistemas operativos con el!

Pero hay que saber lo que es el TTL


El TTL es el numero de "Saltos" que puede dar un paquete dentro de una red hasta ser descartado

Este puede cambiar de OS en OS


Gran parte de los sistemas operativos tienen un TTL predefinido

Como podria ser Windows que siempre usa 128 (Hay algunas excepciones)


Aca dejo una tabla con los TTL de diferentes OSes


Linux: 255/64

MacOS: 64

Windows: 128/32


Ya sabiendo lo que es el TTL y el TTL que usan diferentes sistemas operativos podemos partir con el codigo

Haciendo un ping 127.0.0.1 en nuestra terminal podremos ver el TTL que usamos
