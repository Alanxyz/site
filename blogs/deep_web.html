Que es la Deep Web?

Internet

La Deep Web es conocida por ser ese lugar escondido, oculto de la mirada ajena y la casa de gran parte de los delincuentes

esto... claramente es una mentira del tamaño de una casa


La Deep Web no es nada mas que un monton de URLs imposibles de buscar

Estas URLs no indexadas pueden ser desde videos de YouTube privados hasta Carpetas privadas de Mega, Drive u otros


En estos casos podemos hacer que nuestra pagina no sea indexable de diferentes formas

Ya sea usando Meta tags <meta name="robots" content="noindex">

O deshabilitandolo desde Robots.txt (El cual es un archivo que sirve a los robots para saber que indexar y que no)

pero... Antes de seguir con los diferentes metodos para hacer una pagina no indexable necesitamos saber lo que es un User-Agent


Un User-Agent es un String que nos da informacion del browser, sistema operativo y demas de un usuario.


En Robots.txt podemos especificar el User-Agent que queremos afectar

Se haria en la primera linea

User-agent:

Y se pondria el User-agent (En caso de querer afectar a todos los User-Agents tendremos que poner un * )



Y ya conociendo los metodos mas comunes para hacer una pagina No indexable les hablare de Tor y el .onion


Tor es un proyecto enfocado a mantener los datos de los usuarios protegidos de los gobiernos, ISPs y redes publicas

Usa un sistema llamado Onion Routing

en resumen


El Onion Routing se basa en obtener una peticion de un usuario, pasarla de un computador a otro y asi varias veces hasta que por fin llegamos a la pagina web que se solicite.


cabe aclarar que toda la informacion esta completamente encriptada (excepto los nodos de entrada y salida)

Asi es como podemos cambiar una IP varias veces hasta llegar a una web (lo cual hace que las paginas web se ralenticen)

esto conduce a varios problemas, como podria ser que un nodo modifique tus peticiones e injecte codigo malicioso para obtener datos

(de esto habla mas a fondo Chema Alonso en una conferencia llamada "Owning bad guys and mafia with JavaScript botnets")

Pero no hablamos de eso

Una de las formas mas complicadas de hacer tu pagina no indexable es prenderla en la "deep web"

cuando hablo de la deep web hablo de las tipicas paginas .onion

El cual es una extension usada por Tor para que no sean indexadas por otros buscadores como Google

estas paginas con extension .onion solo pueden ser accedidas desde Tor
