Enumeracion de usuarios en WordPress

Python, web

Hola gente, ahora estoy con unos primos asi que sera corto este blog

Ayer estuve experimentando con un ataque llamado "User Enumeration" en paginas de WordPress (Aprovechando la ruta WP-JSON)

Pero primero, en que consiste la enumeracion de usuarios o User Enumeration?


La enumeracion de usuarios consiste en obtener usuarios (normalmente en plataformas web) usando tecnicas como brute force para generar estos usuarios y probandolo contra sistemas de login, viendo el status code de respuesta de una url u otros.


Ya sabiendo lo que es y como normalmente se hace, veamos como hice el script



Primero importa las librerias sys y requests

Creamos una variable userCount que sume un usuario por cada revision

Y un bucle while, que al entrar a el suma 1 a la variable userCount

Aprovechando eso hace una peticion HTTP por metodo GET al primer valor de una array mas una ruta en donde se almacenan los usuarios(sys.argv)

Asi se veria: https://"sys.argv[1]"/wp-json/wp/v2/users/"userCount"

Obtenemos el valor del JSON y la ponemos en una variable

Ahora se revisa si dio un status code mayor a 400 (Errores del servidor o peticion en si)

Si es asi, solo lo pasamos para evitarlo

Con un else printeo el username en el JSON, ID, la URL del usuario y si es superadmin (Aun que normalmente da un status code de 401 El cual indica que la URL que intenta ser accedida esta protegida)
