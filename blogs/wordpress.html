Enumeracion de usuarios en WordPress

Python, web

Hola gente, ahora estoy con unos primos asi que sera corto este blog

Ayer estuve experimentando con un ataque llamado "User Enumeration" en paginas de WordPress (Aprovechando la ruta WP-JSON)

Pero primero, en que consiste la enumeracion de usuarios o User Enumeration?

La enumeracion de usuarios consiste en obtener usuarios (normalmente en plataformas web) usando tecnicas como brute force para generar estos usuarios y probandolos contra sistemas de login, viendo el status code de respuesta de una url u otros.

Ya sabiendo lo que es y como normalmente se hace, veamos como hice el script



Primero importa las librerias requests que se puede instalar con pip install requests y sys que es Built-in

Revisamos si el usuario paso algun argumento usando la variable argv de sys que devuelve una lista con los parametros que el usuario paso

Si no paso algun argumento, lo informara y cerrara el programa

Despues de esto hacemos una peticion HTTP por metodo GET (Que obtendra la ruta /wp-json/wp/v2/users)

Y revisamos si dio un codigo de status 400 hacia arriba (Errores en la peticion o en el servidor)

Ahora obtenemos el JSON que devolvera la peticion en caso de un codigo de estado 200 (200 OK)

Y usando el mismo JSON que devuelve, es parseado por detras y devuelto en forma de diccionario

Por lo que ahora se pueden obtener los datos usando los square brackets para indexar los elementos
