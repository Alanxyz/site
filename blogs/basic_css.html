Pero primero... Que es CSS?


CSS es un "mecanismo" que le indicara a nuestro navegador como mostrar las cosas


Con este podemos cambiar fuentes, colores y podemos modificar miles de cosas mas

Darle estilo

Ya sabiendo lo que es CSS podemos continuar

Debemos tener un archivo CSS

Lo podemos llamar como queramos


Lo primero es importarlo al archivo HTML

<link rel="stylesheet" href="direccion/de/tu/archivo.css">

Este debe ser puesto en <head>

cabe aclarar que tambien podemos llamar el link con el archivo CSS


Ya llegados a este punto necesitamos saber unas cosas mas

Que son las Classes e IDs?

Estos son atributos que les podemos dar a nuestras etiquetas HTML los cuales podemos buscar y modificar con CSS

Como los podemos agregar a nuestras etiquetas?

<p class="Clase" id="id">Paragraph</p>

Y en CSS podemos acceder a estos valores de la siguiente manera


Clases

.clase {}

IDes

#id {}

Tambien podemos acceder a etiquetas de esta forma

h1 {}

Ahora pasemos a ver atributos basicos de CSS

IDs, clases y tags pueden usar las mismas funciones de CSS pero en el ejemplo usare clases



Aca podemos ver varios atributos

Veamos lo que hacen uno por uno



font-family:

Con este atributo podemos definir una lista de familias

Si la primera fuente no esta disponible se usara la siguiente


color:

Con el atributo "color" podemos definir el color de la etiqueta

En caso de usarlo con un <p> por ejemplo se cambiara el color del texto


background-color:

Con background-color podemos definir el color de fondo


text-align:

Con text-align podemos definir el lugar en donde queramos nuestra etiqueta

Nos permite mover la etiqueta al centro, izquierda y derecha (Estas son sus funciones basicas con el text-align se puede hacer mas)


font-size:

Con font-size podemos definir el tamaño de la fuente
