En este blog aprenderemos a obtener informacion sensible de servidores con una peticion

Pero primero... Que es una peticion HTTP?

Una peticion HTTP es una conexion que hace una maquina a una pagina web

Usuario <-> Servidor

Ya sea para obtener, mandar, modificar o crear datos

Al hacer una peticion el servidor te respondera

Ya sea con el contenido de un archivo u otros datos

Se les llaman metodos y son los siguientes

GET: Obtener

POST: Enviar datos

PUT: Crear o modificar un recurso

DELETE: Elimina

Ahora que conocemos los diferentes metodos podemos hacer lo siguiente



Que hace el script?

El script importa la libreria requests (Tienes que instalarla con pip pip install requests)

Con la libreria hacemos una peticion GET a una pagina web

Creamos una variable la cual almacena esta peticion do_get

Imprimimos el status de la pagina con do_get.status_code

La variable do_get Nos permite acceder a un diccionario llamado headers el cual almacena todos los valores y llaves de los headers con los que nos responda el servidor

Ahora, usamos el diccionario y buscamos una llave usando la funcion .get('Llave')

Cabe aclarar que esta funcion solo es para diccionarios
