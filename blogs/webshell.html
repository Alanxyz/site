Primero... que es una Web Shell?


Una Web Shell es un script (En cualquier lenguaje) que puede ser subido a un servidor web y recibira comandos que despues ejecutara


Normalmente estas Web Shells se crean en lenguajes para web (Como PHP)

Y ejecutan comandos usando los valores de los parametros de una URL

Los parametros se inician usando ? y para definir sus valores se usa =

cabe aclarar que estos valores no pueden estar todos juntos, se separan usando &

Algo como esto: http://example.com/path?primerParametro=1&segundoParametro=2

Ahora que sabemos lo basico, podemos comenzar con el codigo



En primer lugar importamos la libreria Flask, su clase Flask y la funcion request (Se instala con pip install flask)

E importamos os (No se necesita instalar)

Esta libreria sera la encargada de ejecutar los comandos

Ahora definimos app que sera un objeto con el cual podemos correr y manipular las opciones de este script

Para crear una ruta usaremos @app.route(), que es una funcion en donde le pasaremos un parametro

En este caso la ruta que queramos crear (/shell)

Ya con la ruta creada hacemos una funcion shell que definira command en donde se parsearan los parametros y se obtendra el parametro shell

Para ahora redefinir command a os.popen que es la funcion que ejecutara el comando

Despues de ejecutar el comando se retornara el output reemplazando los saltos de linea por <br>

Y finalmente se correra la App con la funcion .run() del objeto app que definimos al inicio
