Parseando RSS con Python

Python, Feed

En este blog aprenderemos a parsear RSS

Pero primero... Que es RSS?

Es un formato de XML para compartir contenido en paginas web

Generalmente se usa para compartir contenido a los usuarios que se suscriben

Normalmente este archivo muestra el titulo de las publicaciones, descripcion y mas informacion

Hoy nos vamos a centrar en obtener los titulos y la URL de todos los posts en una web

Para poder hacerlo necesitaremos una libreria llamada feedparser

Las puedes instalar con pip install feedparser

Empecemos!



Primero importamos la libreria feedparser

Lo segundo es crear una variable

La llamaremos link y sera el encargado de preguntarnos la URL del archivo XML

Despues crearemos una variable llamada feed la cual va a parsear nuestro documento

De ahi crearemos una variable feed_title la cual usara a la anterior variable para buscar el titulo

Ahora creamos una variable llamada feed_entries la cual va a obtener todas las entradas en el archivo

Para poder imprimirlas todas haremos un bucle for que recorra la lista de entradas

Usando el valor .title y .link obtendremos el titulo y el link
